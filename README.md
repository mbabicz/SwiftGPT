## ABOUT THE PROJECT
SwiftUI application that contains OpenAI GPT-3.5 Turbo or GPT 4 chat bot and OpenAI DALL路E 2 system. OpenAI client providing GPT-3.5 is implemented with [this library](https://github.com/alfianlosari/ChatGPTSwift). 
The DALL路E client is provided by [OpenAIKit](https://github.com/MarcoDotIO/OpenAIKit). To use OpenAI you will need to obtain your API from the [OpenAI website](https://openai.com/api/).  <br /> <br />

## INSTALLATION
* Enter your [api key](https://openai.com/api/) in API.swift file
here:
```swift
static let apiKey = "API_KEY"
```
